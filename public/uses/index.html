Stuff I Use

Hardware

Computer

  • OS: Windows 11. I'm not touching Linux until it can play Monster Hunter Wilds on Steam.
  • Processor: Intel Core i5-12600K
  • Graphics:
    • NVIDIA GeForce RTX 3060
    • Intel UHD Graphics 770 (Integrated)
  • RAM: 16GB...could be better, but RAM is so expensive nowadays...

Monitors

  • Samsung 24" LF24T350FHNXZA (black)
  • msi PRO MP243L E14 (black)

Accessories

  • Keyboards:
    • 8BitDo Retro Mechanical Keyboard (N Edition) and 8BitDo Retro 18 Mechanical Numpad (N Edition) with Outemu Lime V3 Tactile switches
    • K780 Multi-Device Wireless Keyboard (dark grey/speckled white) for times when my wife and I want to hang out together in complete silence.
  • Mouse: Logitech M330 Silent Plus (black)
  • Graphics tablet: XP-Pen Fun Deco S (black)

Software

Editors

  • VSCodium - A distribution of VS Code but without any telemetry/tracking
  • Notepad++ - For note-taking
  • Dracula - Editor theme
  • Comic Mono with ligatures - Editor font

Technologies & Frameworks

  • Neocities - Hosting
  • Porkbun - Domain
  • Eleventy - Static site generator
  • Bootstrap - Grid formatting

Web Browsers

  • Firefox - Main browser
  • Ecosia Browser - For testing sites on Chromium browsers

Image Editing

  • Clip Studio Paint - Primary drawing program
  • Photoscape - Extra filters and finishing touches
  • Color Quantizer - Optimization and dithering
  • Aesprite - Pixel art
