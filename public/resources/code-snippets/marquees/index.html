"Old-Fashioned" Marquee with Pure CSS

Example

HTML

<div class="marquee-container" aria-hidden="true">
  <div class="marquee-content">
    <img src="[IMAGE_PATH]" alt="">
    <img src="[IMAGE_PATH]" alt="">
    <img src="[IMAGE_PATH]" alt="">
  </div>
</div>
  • Line 1: aria-hidden is set to true, assuming that the element that will hold the marquee will be decorative. More information about this can be found here.
    • I've seen some sites use marquees to display links to other websites. If this is the case for you, aria-hidden can be removed, and I would recommend adding alt text to the images.

CSS

(Click to show/hide code)
@keyframes marquee {
  0% { transform: translateX(100%); }
  100% { transform: translateX(-150%); }
}

.marquee-container {
  overflow: hidden;
  white-space: nowrap;
  width: 100%;
}

.marquee-content {
  display: inline-flex;
  gap: .5rem;
  animation: marquee 30s steps(400, end) infinite;
}

.marquee-content:hover {
  animation-play-state: paused;
}

.marquee-content img {
  vertical-align: middle;
}
  • Lines 2-3: translateX values ensure that all items in the example "disappear" before cycling through again. These numbers can change depending on how wide the marquee will be and how many items will be in it. Feel free to experiment!
    • For the marquee to scroll from left to right (as opposed to right to left in the example), reverse the numbers. (change 100% to -150%, and change -150% to 100%)
  • Line 9: this width can be any number. I just chose 100% so that it would span the width of this page.
  • Line 14: gap can only be used by flex boxes, and it gives some space between flex elements. This is optional and can be removed.
  • Line 15: 30s can be changed to have the following effects:
    • Lower number = faster scrolling
    • Higher number = slower scrolling
  • Line 15: 400 in steps() can be changed to have the following effects:
    • Lower number = choppier movement
    • Higher number = smoother movement
  • Lines 18-20: Causes the marquee to pause at the user's cursor position when hovered over. This is optional and can be removed.
