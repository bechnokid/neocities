DeviantART 2005-2006

Unable to loosen my nostalgia goggles, I thought I'd recreate the 2005-2006 layout for DeviantART in all most of its glory!

This is the first Status Cafe theme I ever created! I got a few compliments on it, so I thought I'd share it with whoever would like to relive the old days of DeviantART!

I might do the other iterations of the site if I have the inspiration to do so, but I hope you enjoy this one! If you are having issues of any kind or notice any errors in the code, please feel free to email me at bechnokid[at]yahoo.com!

NOTE: This theme is sadly not 100% mobile-friendly due to some limitations while trying to make the theme as close to the original as possible.

Terms of Service(?)

There is a credit block within the HTML code that I would like to be kept there, but that's completely up to you if you want to leave it there or not! The original deviantART theme isn't mine, by any means!! Other than adding a few of my own touches, I'm not the original theme designer!

All I ask is that you don't claim that you made it all on your own!

I also highly recommend saving the image links (found in the HTML and CSS code) on your own server! It's unlikely that I'll move or delete the image links, but better safe than sorry!

Below is a list of variables in the HTML code that you can change, provided with descriptions of their original function.

Since this is just a theme, you can change the variables however you'd like. :3

#deviant-main-header
PROFILE_PICTURE

Self-explanatory!

USERNAME

Self-explanatory! Originally limited to 20 characters on the site.

USERNAME_SYMBOL

Signified what kind of member the user was at a glance.

  • Basic member (~)
  • Subscriber (*)
  • Beta tester (=)
  • Senior member (`)
  • Creative staff (Â¢)
  • Administrator ($)
  • Staff member (+)
USER_TAGLINE

Originally used for users to list their full names, but it's mostly been used as a tagline of sorts in those days. DO NOT LIST YOUR REAL NAME FOR THE LOVE OF--

#deviant-attributes
NOUN

Selected from the following list:

  • Anime Artist
  • Anthro Artist
  • Art Appreciator
  • Art Student
  • Cartoonist
  • Comedy Writer
  • Comment Addict
  • Community Addict
  • Community Whore
  • Critic
  • DA Addict
  • Deviant of Many Talents
  • Deviously Annoying
  • Deviously Deviant
  • deviantART Loather
  • deviantART Supporter
  • Digital Artist
  • Fantasy Writer
  • General Digital Photographer
  • General Fiction Writer
  • General Skinner
  • Gift-Giver
  • Illustrator
  • Lurker
  • Mad Scientist
  • New Artist
  • Oekaki Addict
  • Old Fart
  • Pencil Artist
  • Pixel Artist
  • Procrastinator
  • Romantic Writer
  • Self-proclaimed Genius
  • Shadow Deviant
  • Software Developer
  • Varied Artist
  • Wise Ass
GENDER

Original list included only "Male", "Female", and "Unknown" (I think?)

ACCOUNT_DATE

The date the user created their account.

SUBSCRIPTION_DEADLINE

The date when the user's subscription would end. Some users had the special privilege of having their subscription last until, and I quote, "Hell freezes over" If the user didn't have a subscription, this would not be listed.

PAGEVIEW_NUMBER

The number of visits the user's profile had accumulated. This was so big back then, I kinda miss it!

COUNTRY

Self-explanatory!

STATUS_IMAGE, STATUS_ALT, and STATUS_TOOLTIP

An image selected from a list of emojis (then known as "emoticons"). You can use regular emojis for this one, but if you would like to make the page as authentic as possible, a list of deviantART's emojis is listed under deviantART Emoticons along with their original tooltip text.

DISCORD_USERNAME

Originally used for users to list their handle on AIM, MSN Messenger, etc. I figured putting Discord here would be appropriate until I find more IM apps that are used a lot!

#deviant-links
WEBSITE_BTN_LINK, WEBSITE_BTN_IMG, and WEBSITE_BTN_ALT_TEXT

If a user had a website, then they had the option to put their button here. Be sure to replace the image link with your website button.

WEBSITE

Similar to the website button, you can just put in the link of your website (or another one entirely) here.

EMAIL

Self-explanatory!

#deviant-stats
DEVIANT_USER_STATUS

Provided a description of the type of member that the user was.

  • Member (~)
  • DeviantART Subscriber (*)
  • Official Beta Tester (=)
  • Senior Member (`)
  • Creative Staff (Â¢)
  • Administrator ($)
  • deviantART Staff (+)

The other numbers in this list can be changed to what ever you'd like. For authenticity's sake, numbers have a comma, e.g. 1,000.

#deviant-about
DEVIANT_ID_IMG and DEVIANT_ID_ALT_TEXT

This differs from the profile picture. Here, users can show a larger image of themselves.

#deviant-list

Think of this list as the offical deviantART survey/quiz. You can list all, some, or none of these questions! I never really understood what "Skin of choice" meant...

Variables

Below is a list of variables in the CSS code that you should change!

NameDescription
STATUS_CAFE_USERNAME

Self-explanatory. This is used to hide the "Subscribe with Atom" section on your page.

USERNAME_SYMBOL

Should match the same USERNAME_SYMBOL that you choose for your HTML code.

Similar to forum signatures. Unfortunately, due to the limitations of pseudo classes, this can only be in plain text, and no HTML can be used.
