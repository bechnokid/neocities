deviantART '06

Unable to loosen my nostalgia goggles, I wanted to recreate the 2005-2006 deviantART layout in all most of its glory!

The first Status Cafe theme I've created! I got a few compliments on it, so I thought I'd share it for those who would like to relive the old days of DeviantART!

I might do other iterations of the site if I have the inspiration to do so, but I hope you enjoy this one in the meantime! If you are having any issues of any kind or happen to notice errors in the code, don't be afraid to shoot me an email at bechnokid@yahoo.com!

Terms of Service

There is a credit block within the HTML code that I would like to be kept there, but that's completely up to you if you want to leave it there or not! All I would ask is that you claim you made this on your own.

I also highly recommend saving the image links found in the code into your own server! It's unlikely that I'll move or delete the image links, but it's better to be safe than sorry!

Notes

As you might have guessed from the preview, there are a lot of customizable variables in this theme. They're all technically optional, but most of them are mandatory for the "authentic" look.

Explanations for each one will be provided below.

Main Banner

USERNAME_SYMBOL
signified what kind of member the user was at a glance, with the most basic member being "~". Subscribed members had "*" and beta testers had "=". All other users were staff members or administrators with symbols within this range (`, ¬¢, $, +).
USER_TAGLINE
formerly used to list full names, but can be used as a tag line. Don't ever list your real name.

SUBSCRIPTION_DEADLINE
the date when the user's subscription would end. Some users had the special privilege of having their subscription last until, and I quote, "Hell freezes over" If the user didn't have a subscription, this would not be listed.
NUM_OF_PAGEVIEWS
the number of visits the user's profile had accumulated. This was a big deal back then, I kinda miss it!
"is currrently -insert emoji-"

IM_MESSENGER
originally used for users to list their handle on AIM, MSN Messenger, etc. You can still do the same with whatever IM service you use now. For example, if you use Discord, then this line would be "is a Discord user; ...". I think there was a way to put multiple IM services here, but I need to do more research.
IM_MESSENGER_USERNAME
the username for whatever IM service you use.
DEVIANT_USER_TYPE
a description of the type of member that the user was. The basic one was "Member", subscribers would be "DeviantART Subscriber", and beta testers would be "Official Beta Tester".

Devious Information

Think of this entire list as deviantART's own survey. You can list all, some, or none of these questions! I never really understood what "Skin of choice" meant...maybe it was for WinAmp?

Extra Boxes/Widgets

If you'd like to add an extra box, you can copy and paste the following code right above the credits block.

<div class="deviant-container">
  <div class="deviant-header">
    <h2>>WIDGET_TITLE</h2>
  </div>
  <div class="deviant-content">
    <p>WIDGET_CONTENT</p>
  </div>
</div>

Statuses

For the section containing the statuses, you would need to go into the CSS and change the variables.

USERNAME_SYMBOL
matches the symbol chosen in the banner
SIGNATURE (optional)
similar to forum signatures. Unfortunately, due to the limitations of pseudo classes, this can only be in plain text, and no HTML can be used. You can remove signatures by deleting the &:after block in the .status-content block.
